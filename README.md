🚀 Disaster Management System

A Flask-based web application designed to coordinate emergency services, manage resources, and assist affected individuals during natural or man-made disasters.

📌 Overview

This project provides an integrated platform that enables real-time interaction between users and authorities, ensuring efficient allocation of critical resources such as hospitals, ration supplies, and volunteers. It centralizes disaster-related information and streamlines response operations.

🎯 Features
👨‍💼 Admin Dashboard
Manage disasters, hospitals, volunteers, and resources
Monitor user requests and system activity
🧑‍🤝‍🧑 User Module
Request help (food, medical aid, shelter)
View available resources and emergency contacts
🏥 Resource Management
Track hospitals and ration shops
Monitor availability and allocation
🙋 Volunteer System
Register and manage volunteers
Assign tasks during emergencies
📍 Disaster Tracking
Add and monitor ongoing disasters
Maintain centralized disaster information
🛠️ Tech Stack
Backend: Flask (Python)
Frontend: HTML, CSS, JavaScript
Database: SQLite / MySQL
Other: REST APIs, Session Management

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/RedFire4444/Disaster-Management-System.git
cd Disaster-Management-System
2️⃣ Create virtual environment
python -m venv venv
3️⃣ Activate environment
Windows:
venv\Scripts\activate
Mac/Linux:
source venv/bin/activate
4️⃣ Install dependencies
pip install -r requirements.txt
5️⃣ Run the application
python app.py
6️⃣ Open in browser
http://127.0.0.1:5000
